## 🚀 ¿Qué es este sistema?

El **Sistema de Monitoreo de Serenazgo** es una solución profesional desarrollada para optimizar el control, supervisión y registro de operaciones de unidades vehiculares en tiempo real.

Está diseñado para entidades municipales y organizaciones de seguridad que requieren **precisión, control y toma de decisiones basada en datos**.

---

## 🎯 Problema que resuelve

❌ Registro manual desordenado
❌ Falta de control de unidades
❌ Pérdida de información operativa
❌ Procesos lentos y sin trazabilidad

✅ Este sistema digitaliza y centraliza toda la operación

---

## ⚡ Solución

✔️ Control total de unidades en tiempo real
✔️ Registro estructurado de KM, A.P y P.O
✔️ Reportes automáticos en Excel
✔️ Interfaz moderna, rápida e intuitiva
✔️ Automatización de procesos operativos

---

## 🧩 Funcionalidades Clave

### 🚙 Gestión de Unidades

* Control de Pickups y Autos
* Selección masiva o individual
* Estado en tiempo real

### 📊 Registro Operativo

* Kilometraje (KM)
* Auxilio Público (A.P)
* Personal Operativo (P.O)

### ⏱️ Control de Turnos

* ☀️ Día
* 🌙 Noche

### 📈 Reportes Inteligentes

* Exportación automática a Excel
* Historial de registros
* Backup automático

### 🛠️ Herramientas Integradas

* 📸 Captura de pantalla
* 📷 Gestión de cámaras
* 📢 Megáfono
* 🖨️ Impresión de reportes

## 📦 Instalación

```bash
git clone https://github.com/tuusuario/serenazgo-system.git
cd serenazgo-system
pip install -r requirements.txt
python main.py
```
